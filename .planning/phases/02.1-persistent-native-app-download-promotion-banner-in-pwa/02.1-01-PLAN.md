**AppStoreBanner.jsx — Full rewrite with these changes:**

1. **Dismissal: sessionStorage instead of localStorage.** Replace `localStorage.getItem/setItem(DISMISS_KEY)` with `sessionStorage.getItem/setItem(DISMISS_KEY)`. Remove the `REDISPLAY_DAYS` constant and all date math — just store a simple flag `'true'`. On mount, check `sessionStorage.getItem(DISMISS_KEY)` — if truthy, stay hidden. On dismiss, `sessionStorage.setItem(DISMISS_KEY, 'true')`.

2. **Show on ALL platforms including desktop.** Remove the `if (platform === 'desktop') return;` guard in the useEffect. Banner shows on ios, android, AND desktop.

3. **Desktop layout: QR code instead of store button.** When `platform === 'desktop'`, render a QR code image that encodes the URL `https://levelupcincinnati.org/download.html` (the existing landing page with platform auto-detection). Generate the QR code using a free API: `https://api.qrserver.com/v1/create-qr-code/?size=80x80&data=https://levelupcincinnati.org/download.html`. Display the QR code image (80x80px, white background, rounded corners) where the CTA button would be on mobile. Add small text below or beside it: "Scan to download".

4. **Mobile layout: Keep platform-specific button.** iOS shows Apple icon + "Get App", Android shows Shop icon + "Get App" — same as current behavior.

5. **Styling per user decisions:** Full-width slim bar, fixed to top, Primary Blue (#18264E) background, white text, Coral (#F15F5E) CTA button. Keep existing `zIndex: 1500`, slide-down animation, box-shadow, close button, logo, and overall layout structure. Keep dark mode support (`theme.palette.mode === 'dark' ? '#1a1a2e' : '#18264E'`).

6. **Messaging:** Use migration-encouraging copy. Title: "Level Up Cincinnati". Subtitle: "Switch to our new app for the best experience" (slightly more migration-focused than current "Get the app for the best experience").

7. **Remove the 1500ms delay timer.** Show banner immediately on mount (no setTimeout for visibility). The useEffect just checks sessionStorage and sets visible if not dismissed.

8. **Also clear the old localStorage dismiss key on mount** to reset any existing 14-day cooldowns: `localStorage.removeItem('appStoreBannerDismissed')` — do this once then the sessionStorage logic takes over.
